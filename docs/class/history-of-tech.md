# 科技简史

这门课适合作为理工科课程之外的视野补充，核心不是背年代，而是理解“技术如何塑造社会、社会又如何反过来选择技术”。

!!! info "课程速览"
    - 学习重点：技术演进逻辑与关键转折点
    - 课堂形式：讲授 + 讨论
    - 适合人群：希望补齐科技史与跨学科视角的同学

## 课程看点

- 从经典技术节点出发，梳理“发明-应用-扩散-影响”的完整链路。
- 讨论技术发展背后的经济、制度与文化因素，而不只停留在时间线。
- 对做科研、工程实践和写作都很有帮助，能提升问题分析的纵深感。

## 学习建议

- 课前快速了解当节主题的历史背景，上课会更容易跟上讨论。
- 建议结合自己的专业方向做小型延伸阅读，收获会明显更大。
- 这门课的价值在于“建立框架”，不是“记住所有细节”。
